3. Largest N¶

  • df.nlargest(n, columns='column_name') : Selects the n rows with the largest values in the specified single column.
  • df.nlargest(n, columns=['col1', 'col2']) : Selects the n rows with the largest values, prioritizing col1 first, then col2 for ties, and so on for multiple specified columns.
  • df.nlargest(n, columns='column_name', keep='first') : Selects the n rows with the largest values, keeping the first occurring rows if there are ties.
  • df.nlargest(n, columns='column_name', keep='last') : Selects the n rows with the largest values, keeping the last occurring rows if there are ties.
  • df.nlargest(n, columns='column_name', keep='all') : Selects the n rows with the largest values, keeping all rows if there are ties that would result in more than n rows.

5. Rank¶

The rank() method in Pandas assigns a numerical rank to each value within a Series or DataFrame based on its position in the sorted order of values. It's particularly useful for understanding the relative standing of each data point, such as identifying the top-performing items or segmenting data into quartiles. The method offers various strategies to handle tied values (e.g., assigning the average rank, minimum rank, or sequential rank based on appearance) and options for handling missing data, providing flexibility for different analytical needs.

  • df['column_name'].rank() : Assigns a rank to each value in the Series, with ties receiving the average rank (default 'average' method).
  • df['column_name'].rank(method='average') : Assigns average rank to values that are equal (ties).
  • df['column_name'].rank(method='min') : Assigns the minimum rank to values that are equal (ties).
  • df['column_name'].rank(method='max') : Assigns the maximum rank to values that are equal (ties).
  • df['column_name'].rank(method='first') : Assigns ranks in the order of appearance for values that are equal (ties).
  • df['column_name'].rank(method='dense') : Assigns ranks without gaps, assigning the same rank to equal values.
  • df['column_name'].rank(ascending=False) : Assigns ranks in descending order, where the largest value gets rank 1.
